Build on Flare

Unlock the power of decentralized data and build scalable, real-world applications on Flare.

Why Build on Flare?

Flare provides a scalable, interoperable, and developer-friendly ecosystem to create dApps with real-world data integration. Whether you're building in **DeFi, NFTs, supply chain, gaming, or identity solutions**, Flare's infrastructure ensures security, efficiency, and cost-effectiveness.

Its **Ethereum Virtual Machine (EVM) compatibility** allows easy migration of existing Solidity-based smart contracts while **Flare Time Series Oracle (FTSO)** and **State Connector** unlock access to secure off-chain data.

How to Get Started

1. Set Up Your Environment

Install **Node.js**, **Truffle**, and **Hardhat**. Use **Metamask** to connect to the Flare network.

2. Deploy Your First Smart Contract

Write Solidity contracts and deploy them using Hardhat or Remix IDE.

3. Integrate Data from Flare

Use the **Flare Time Series Oracle (FTSO)** or **State Connector** to integrate off-chain data.
